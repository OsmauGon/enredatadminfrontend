Paso 1: Creación 
Ejecutamos npx-create-next-app@latest nombreDELproyecto. Inmediatamente nos pregunta si queremos hacer la configuración recomendada de Next o customizada, a lo que elegimos customizar.

√ Would you like to include AGENTS.md to guide coding agents to write up-to-date Next.js code? ... No / Yes (elegimos NO)
De esta manera sabemos que estamos creado y con que configuracion

Next en un framework sobre React que da estructura, backend integrado, routing automatico y optimizacion lista. En otras palabras React solo en una libreria y Next es el entorno completo para construir apps reales
Con Next ya renemos integrado el routing (react-router-dom), el fetch de datos y otros. La estructura de Next se basa en carpetas, entonces por ejemplo cada ruta (loginPage, dashboardPage) esta dentro de su propia carpeta: login/page.tsx y dashboard/page.tsx

Paso 2: Primeros archivos (para el dia de la fecha haremos un login minimo y funcional)
Una vez todo el proceso de creacion completado tendremos la carpeta del proyecto y una de sus carpetas es SRC y dentro APP y dejar creadas las carpetas LOGIN y DASHBOARD, cada una con un archivo page.tsx

ambos page.tsx deben empezar con "use client" y sus funciones de componente se exportar si o si de la siguiente manera: 
export default function nombreDEfuncion (){}

ya podemos ejecutar npm run dev para levantar el servidor y debemos ir en el navegador a localhost:3000/login y probar loguearnos
